## Project Setup
1. Install node 12+ version
2. Generate `.env` from `.env.example`.

### Install dependencies
```bash
$ npm install
```

### Edit data for calculating
#### for default here data from task description
```
./input.json
```

### Start a project
```bash
$ npm run start
```

#### Result of calculating display in console

### Start a Unit Test
```bash
$ npm run test
```
